# worklet-example

Browser api worklet example
[API tutorial](https://developer.mozilla.org/en-US/docs/Web/API/PaintWorklet)

## Plugins
* webpack (v4.40.2)
* babel (v7)
* es2015

## Usage

*1) npm install*

```bash
npm install
```

*2) start*

```bash
npm run start
```

*3) module bundling*

```bash
npm run dev
```
